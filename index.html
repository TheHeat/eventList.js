eventList.js Demo

eventList.js is a super-simple to use jQuery function for displaying a list of forthcoming events on your page.

Easy peasy

Getting eventList.js up and running is really easy.

  • Find your calendar ID in the calendar settings page. The calendar must be publicly available.
  • Generate a Google Calender API key
  • Include the script in your head (you’ll need jQuery too obvs).
  • Extend the target object .eventList();, passing in the ID for a publicly available Google calendar you would like to display.
<div id="target-div"/>
<script>
	$(document).ready(function(){

		args = {
			key: 'YOUR_GCAL_API_KEY',
			calID: 'YOUR_GOOGLE_CALENDAR_ID@group.calendar.google.com',
			maxResults: 12
		}

		$('#target-div').eventList(args);
	});
</script>
